# Power BI Projects Repository

This repository contains a collection of Power BI dashboards and data analysis projects developed to explore datasets, generate insights, and visualize key performance indicators through interactive reports.

The projects demonstrate skills in data cleaning, data modeling, DAX calculations, and dashboard design using Microsoft Power BI.

## Repository Contents

This repository includes multiple Power BI projects covering different domains such as:

- Business analytics

- Sales and revenue analysis

- Customer insights

- Data visualization and reporting

Each project contains Power BI report files (.pbix) along with datasets where applicable.

## Tools & Technologies

The following tools and technologies were used to build these projects:

- Microsoft Power BI

- Power Query

- DAX (Data Analysis Expressions)

- Microsoft Excel / CSV datasets

- Data Modeling

- Data Visualization

## Key Features Demonstrated

These projects demonstrate:

- Data cleaning and transformation using Power Query

- Creating interactive dashboards

- Designing data models and relationships

- Writing DAX measures and calculated columns

- Building KPIs and business metrics

Creating filters, slicers, and drill-down reports

Visual storytelling through charts and visuals
